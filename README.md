# Projeto-ecommerce





Nome: Matheus Spilmam



Atividade de Banco de Dados feita no SENAI Curso de Técnico em Desenvolvimento de sistemas. 

Contextualização: Uma empresa de e-commerce em expansão enfrenta dificuldades para gerenciar dados de clientes, produtos e pedidos, atualmente armazenados em planilhas e arquivos simples. O aumento no volume de transações tem gerado falhas de integridade, dificuldade na geração de relatórios e ausência de segurança e backup. A equipe de desenvolvimento precisa projetar um banco de dados robusto e escalável, que organize essas informações, permita consultas eficientes e implemente procedimentos de segurança e recuperação de dados.

Neste repositório contém todos os modelos pedidos, conceitual, lógico e físicos.
